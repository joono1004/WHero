# World in Hero — Game Vision

역사 영웅과 여러 부대를 운용해 절차적으로 생성되는 세계를 정복하고, 일부 전력을 계승하며 더 큰 세계로 진출하는 웹 기반 턴제 전략 게임이다.

## 확정된 원칙

- Pointy-top Hex 경계는 이동 동선을 읽을 수 있도록 표시한다.
- 지형 그래픽은 Hex마다 단절된 조각처럼 보이지 않도록 주변 지형과 연결한다.
- 이동과 일반 전투는 세계지도 위에서 진행한다.
- 일반 공격은 짧은 공격·피격 애니메이션으로 표현한다.
- 특별한 영웅 기술은 선택적으로 별도 연출 레이어를 사용할 수 있다.
- 영웅이 없는 일반 부대도 이동과 전투가 가능하다.
- 영웅은 부대 또는 도시에 배속되어 전문 보너스를 제공한다.
